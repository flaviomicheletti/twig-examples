Escape single lines as strings:

{% verbatim %}{{ '{% set foo = "bar" %}' }}{% endverbatim %}

Use autoescape to escape multiple lines, optionally specifying a syntax (see docs):

{% verbatim %}{% autoescape %}
    {% if foo != "bar" %}
        {% set foo = "bar" %}
    {% endif %}
{% endautoescape %}
{% endverbatim %}

Use verbatim to treat multiple lines as raw text:

{% verbatim %}{% verbatim %}
    {% if foo != "bar" %}
        {% set foo = "bar" %}
    {% endif %}
{% endverbatim %}
{% endverbatim %}
